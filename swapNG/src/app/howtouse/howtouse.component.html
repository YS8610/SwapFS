looks_oneWhat is needed?

You will need to have a telegram account with username defined before you use this web app. To get started, you will need to get the chatid from the smart_toy telegram bot by entering /chatid which will return a string of number which will identify you.

looks_twoAdding Swap Request

To submit a lesson swap request, you have to click the add_circle. Please use the verify function of the form to verify that you have entered the correct chatid. You will received a message from the bot if your chatid is entered correctly. If the chatid is entered wrongly, then you will not be able to received any notification when interested party try to contact you. This is because the bot identifies you by chatid and is the messenger between you and the interested party.

looks_3Contacting the poster

To contact the swap request poster, you have to click message button in the home home page. When you submit the message, the telegram bot forward it to the poster. So please make sure you leave a proper contact detail behind.

looks_4Telegram Botsmart_toy Command
  • /chatid - give the chatid which the bot will use to identify userComment
  • /swap - return all a list of your swap requests
  • /delete - return a list of button for you to delete your existing swap request (This is the only way to delete)
